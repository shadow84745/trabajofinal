Acerca de

Reballsystem es un centro técnico especializado para todo tipo de fallas electrónicas.

  • Servicio técnico especializado

    Ofrecemos un servicio técnico especializado para equipos electrónicos, ya sean laptops, equipos de mesa, televisores, consolas de videojuegos (Xbox, PlayStation, Nintendo), entre otros. Realizamos todo tipo de reparaciones, además de mantenimientos preventivos y correctivos a equipos de cómputo.

  • Reballing, Rework y Reflow

    El reballing es una técnica de reparación que consiste en resoldar algún componente, generalmente el chip gráfico o GPU. Hacemos uso de máquinas especializadas para este proceso, llamada “Rework System”, que es capaz de rehacer la soldadura siempre que esta sea de tipo BGA.

  • Servicio técnico de drones DJI

    Servicio Técnico para Drones de distintas marcas o referencias, nos especializamos en drones DJI.

  • Reparación de pantallas

    Contamos con tecnología única en el Huila, para reparar pantallas de televisores y monitores.

Reballing

El reballing es una técnica de reparación que consiste en resoldar algún componente, generalmente el chip gráfico o GPU. Hacemos uso de máquinas especializadas para este proceso, llamada “Rework System”, que es capaz de rehacer la soldadura siempre que esta sea de tipo BGA.

Computadores

Ofrecemos un servicio técnico especializado para equipos de cómputo, ya sean laptops o equipos de mesa (ASUS, HP, ACER, Lenovo, MacBook, iMac, entre otros). Realizamos todo tipo de reparaciones, además de mantenimientos preventivos y correctivos.

Televisores

Ofrecemos un servicio técnico especializado para televisores de distintas marcas (LG, Samsung, Sony, entre otros.). Realizamos todo tipo de reparaciones, además de mantenimientos preventivos y correctivos, además contamos con tecnología única en el Huila para reparación pantallas.

Drones DJI

Servicio Técnico para Drones de distintas marcas o referencias, nos especializamos en drones DJI.

Consolas

Ofrecemos un servicio técnico especializado para consolas de videojuegos (Xbox, PlayStation, Nintendo, entre otros). Realizamos todo tipo de reparaciones, además de mantenimientos preventivos y correctivos.

Grabado PCB

Ofrecemos el servicio de grabado de PCB. Este proceso lo realizamos haciendo uso de una maquina CNC, con hasta máximo dos capas (Top y Bottom). Si usted ya tiene el proyecto diseñado nos puede hacer llegar los archivos Gerber, o nosotros le colaboramos con el diseño y la realización del mismo.
